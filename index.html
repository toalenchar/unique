01 — About

Built by someone
who had to start over.

Waverider Financial Strategies is the product of self-directed learning, discipline, and a genuine obsession with how markets move. After release in January 2025, I committed to building technical skills from scratch — teaching myself Python, algorithmic backtesting, and quantitative analysis while developing real trading systems.

This isn't a company built on credentials. It's built on curiosity, consistency, and the belief that understanding the math behind markets is accessible to anyone willing to put in the work. I'm that person.

02 — Projects

Systems I've Built

01

EMA Acceleration System

Detects the moment the 8-period EMA changes acceleration — the second derivative of price momentum. Built in ThinkScript for live ThinkorSwim charts with dynamic bull/bear arrows.

ThinkScript ThinkorSwim Technical Analysis
02

VWAP Reversion Backtester

Python system that backtests mean reversion trades off the ±2σ VWAP bands, confirmed by EMA slope alignment across the 8, 13, and 21 EMAs. Pulls live data via the Schwab API.

Python Schwab API Quantitative Finance
03

EMA Slope Alignment Engine

Detects the precise moment EMA slopes align across three timeframes using first and second derivatives. Includes full backtesting framework with win rate, Sharpe ratio, and profit factor analytics.

Python pandas NumPy
